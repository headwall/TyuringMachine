Данное приложение реализует машину Тьюринга

(!) У Вас должен быть установлен Python 3

1. Распаковать архив
2. Открыть командную строку
3. Зайти в директорию с проектом
4. Выполнить команду "python startServer.py"
5. Открыть адрес http://localhost:8000
6. Ввести данные согласно инструкции
7. Нажать кнопку "Отправить"
